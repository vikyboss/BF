# BF
An implementation of an esoteric programming language called Brainf***

# Example
An example test code from language wikipedia page used here for testing:
```
$ echo "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++." | ./BF /dev/stdin
```
