# BF
An implementation of an esoteric programming language called Brainf***

# Example
An example test code for printing "Hello World!" from the language wikipedia page is used here for testing:
```
$ ./BF /dev/stdin <<EOF
++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.
>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.
EOF
Hello World!
```
